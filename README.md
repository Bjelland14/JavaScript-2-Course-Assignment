This is a front-end social media application built with TypeScript and Vite.

The application allows registered users to create, edit, delete and browse posts, as well as follow and unfollow other users.

---

##  Features

- Register new user
- Login user
- View all posts (Feed)
- View single post
- Create post
- Edit own post
- Delete own post
- View another user's posts
- Follow / Unfollow users
- Search posts
- View own profile

---

##  Tech Stack

- TypeScript
- Vite
- ES6 Modules
- Noroff Social API
